# -Sign-Language-Recognition-System

A Python-based Sign Language Recognition System that detects hand gestures from A-Z and converts them into text and voice output. This project is designed to help speech and hearing-impaired individuals communicate more easily using hand gestures.

## Features

- Detects hand gestures from A-Z
- Converts recognized gestures into text
- Provides voice output for detected gestures
- Uses image processing for hand gesture detection
- Supports real-time gesture recognition
- Helps improve communication for people who are unable to speak or hear

## Technologies Used

- Python
- OpenCV
- Keras / TensorFlow
- NumPy
- Image Processing
- Text-to-Speech Library

## Project Objective

The main objective of this project is to create an assistive communication system for deaf and speech-impaired individuals. The system detects hand gestures, recognizes the corresponding alphabet, and converts it into readable text and voice output.

## How It Works

1. The camera captures the hand gesture.
2. Image processing is applied to detect and preprocess the gesture.
3. The trained model predicts the alphabet from A-Z.
4. The predicted alphabet is displayed as text.
5. The system generates voice output for the detected gesture.

## Applications

- Assistive communication for deaf and speech-impaired individuals
- Educational tool for learning sign language
- Human-computer interaction
- Smart accessibility systems
